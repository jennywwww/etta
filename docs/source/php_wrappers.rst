etta.php\_wrappers module
=========================

Python wrappers for PHP functions that are documented here: 
https://exofop.ipac.caltech.edu/tess/Introduction_to_ExoFOP_php_functions.php

Note that this module can also be accessed directly from the ``etta`` package. That is, the functions in this module can be imported as

.. code-block:: python

  from etta.php_wrappers import func_name

or as

.. code-block:: python

  from etta import func_name
